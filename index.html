노토 산스란?

구글과 어도비가 합작으로 만든 웹폰트로, 자세한건 이 링크를 참고해 주세요. 현재 보고 계신 글자들이 바로 노토 산스 한글입니다.

웹용으로 경량화된 노토 산스 한글

이 프로젝트는 @uyeong님의 글과 깃헙 저장소를 기반으로 만들어졌습니다. 폰트 파일에서 자주 쓰이는 글자만 뽑아서 용량을 줄인 버전입니다.

사용법

AWS S3를 CDN 삼아서 서빙하고 있습니다.

AWS S3를 개인 계정으로 사용하고 있었으나, 비용 문제로 jsdelivr로 서빙하게 되었습니다. 안정성이 보장된 CDN이 아니므로, 자체적으로 서빙하시는걸 권장해 드립니다.

HTML에서:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/theeluwin/NotoSansKR-Hestia@master/stylesheets/NotoSansKR-Hestia.css" integrity="sha384-zGzb8nSrN9lm5PPcfztVC5DjjTs5sVStV85IH8x51fM9yIwmI+Uvh3RZDJlrpPWQ" crossorigin="anonymous">

CSS에서:

@import url(https://cdn.jsdelivr.net/gh/theeluwin/NotoSansKR-Hestia@master/stylesheets/NotoSansKR-Hestia.css);

설정 후, 아래와 같이

.noto {
  font-family: 'Noto Sans Korean', sans-serif;
}

등의 방법으로 font-familty로 지정하면 사용할 수 있습니다.
